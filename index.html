Book In Pop Out

Book In Pop Out was the main project for the subject "Data Structures". It is a three tier Java application using a mobile client (made with Java, for Android) as frontend, Java servlets as backend, and MySQL for the database. It allows clients the creation of virtual "priority queues" for the services that require them, avoiding agglomerations of people in large queues waiting to be attended. Besides, it uses concurrency to handle each client's turn. Here are the links to the Github repositories for the client and the backend.
